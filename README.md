# zoheirbachir.github.io

Personal portfolio of **Zoheir Bachir** — Developer & Data Scientist (Oran, Algeria).

Live at **https://zoheirbachir.github.io**

Static site (HTML / CSS / vanilla JS), deployed with GitHub Pages. No build step.

## Structure
- `index.html` — page content
- `styles.css` — dark theme, layout
- `script.js` — project filtering + scroll reveals
- `favicon.svg` — ZB monogram

## Editing
Edit the files, then:

```bash
git add -A
git commit -m "Update portfolio"
git push
```

Changes go live within a minute.
